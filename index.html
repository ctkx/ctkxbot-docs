Moderation Tools

Guild / Server Setup

The .mod commands require some setup which can only be complete by the Server/Guild Owner.
First, a Logging Room must be set. It’s recommended to use a room which only your Moderators can see.
Get the room’s ID , and send the command: .set_logs_room <room ID>

When the logging room is set, you also need to set which roles in your server are allowed to use the .mod_roles command:

Command Additional Input Output
.mod_roles list Lists Active Mod Roles
.mod_roles add "Role Name" Adds “Role Name” to Bot Mod Roles
.mod_roles remove "Role Name" Removes “Role Name” from Bot Mod Roles

Alert words

The alertwords function allows you to have the bot watch for when certain words are posted.
There are 3 types of triggers. All messages containing a trigger word will be logged to the Logging Room that you set.
The following actions will be taken based on the type of trigger:

Trigger Type Outcome
silent No warning is issued and the user does not know about the alert.
warn The message is responded to with a public warning, If the user gets enough warnings, they are banned.
ban The message is responded to with a public warning, And the user is immediately banned.

To add , remove, or list alert words:

Command Additional Input Output
.mod add alertword type alertword=TriggerWord Adds TriggerWord as the specified type of alert
.mod remove alertword alertword=TriggerWord Removes TriggerWord from any alert type
.mod list alertword type Lists trigger words of the specified type.
Use this only in your moderator/logging channel,
The response will not be obscured.

Auto Replies

The autoreplies function makes the bot watch the rooms in your server for certain words. When a valid word is detected, the bot sends a randomly selected response from the corresponding reply group.
Any user can trigger an autoreply, but only moderators can add/remove them.

To add or remove trigger words:

Command Additional Input Output
.mod add autoreply trigger triggerword=.testword replygroup=testreplygroup Adds ".testword" to the testreplygroup group.
.mod remove autoreply trigger triggerword=.testword replygroup=testreplygroup Removes ".testword" from the testreplygroup group.

To add or remove replies:

Command Additional Input Output
.mod add autoreply reply replygroup=testreplygroup replybegin=message text here =replyend Adds "message text here" to "testreplygroup".
All text between replybegin= and =replyend will be saved.
There can be multiple lines of text.
.mod remove autoreply reply replygroup=testreplygroup replybegin= message text here =replyend Removes message text here from testreplygroup group.

To list corresponding trigger words and reply groups

Command Additional Input Output
.mod list autoreply trigger replygroup=testreplygroup Lists all trigger words for testreplygroup
.mod list autoreply replygroup triggerword=.testword Shows which group is triggered by .testword
